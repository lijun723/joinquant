## 🚀 快速开始

1. 打开 [joinquant.com](https://www.joinquant.com) → 微信扫码登录
2. 进入 **研究平台** → **新建策略**
3. 将任意 `.py` 文件的 **全部内容** 复制粘贴到策略编辑器
4. 在右侧设置回测参数：

   | 参数 | 建议值 |
   |------|--------|
   | 回测区间 | `2020-01-01` 至 `2024-12-31` |
   | 初始资金 | `1,000,000` |
   | 基准 | `000300.XSHG`（沪深300） |
   | 佣金滑点 | 使用默认值（买入万3+卖出万3+千1印花税） |

5. 点击 **「运行回测」**

### 独有特性

| 脚本 | 独有特性 |
|------|---------|
| `双均线趋势` | 金叉启动点1.5倍权重奖励，死叉2倍权重强制卖出 |
| `动量突破` | 量比辅助确认（放量=真实突破），月度调仓降交易成本 |
| `布林带均值回归` | 带宽过滤（只做高波动股票），止盈4%+止损7% |
| `反转策略` | **内置指数择时保护**（沪深300在MA60下方时空仓），止盈8%+止损8% |
| `多因子评分` | 5因子横截面归一化，权重可调，带ATR风控 |
| `指数择时+风控` | **双重止损**（固定-8% + 移动止损从最高点回撤-12%） |

## 📊 IC 分析脚本

`h300_因子IC分析.py` — **【研究环境运行】**

分析PE/PB/ROE/换手率与未来20日收益的 Spearman 秩相关性，输出：

- 每个因子的 **平均IC**（方向+强度）和 **ICIR**（稳定性）
- IC > 0 的月份占比（胜率）
- **IC时序折线图** + **箱线图**
- **五组分层收益**（G1低值组 vs G5高值组的多空收益差）
- **因子排名**（按 `|平均IC| × |ICIR|` 综合评分）
- 策略建议

> ⚠️ 只能在 **研究环境（Research Notebook）** 运行，不是回测策略。
> 需加 `from jqdata import *`。

## ⚠️ 重要注意事项

### 1. JoinQuant 新版引擎（jqboson）避坑

| ❌ 不要这样做 | ✅ 正确做法 |
|-------------|-----------|
| `set_commission()` | 使用 JoinQuant 默认佣金（万3） |
| `set_slippage()` | 使用默认千1滑点 |
| `get_current_data()[stock].close` | `attribute_history(...)['close'].iloc[-1]` |
| `def func(context, data):` | `def func(context):` |

### 2. 运行环境区分

| 环境 | 用途 | import |
|------|------|--------|
| **回测策略** | 6个策略脚本 | 函数内置，无需 import |
| **研究环境** | IC分析脚本 | 需要 `from jqdata import *` |

### 3. 通用风控（所有策略共用）

- 停牌股票不下单
- ST股票不买入
- 科创板（`688xxx`）排除
- 等权持仓，无单票过度集中

### 4. 回测 vs 实盘差异提示

- 默认滑点千1，实盘建议按千2-千3测试敏感性
- 大资金 `order_value` 有冲击成本，需拆分
- 停牌复牌首日的涨跌停无法模拟
- JoinQuant 自动处理分红送股复权

## 🔧 参数调优建议

### 按市况选策略

```
如果认为接下来是：    →  建议优先跑：
─────────────────────────────────
牛市 / 趋势上涨      →  双均线趋势 或 动量突破
震荡 / 横盘整理      →  布林带均值回归
超跌 / 反弹行情      →  反转策略
不确定 / 均衡配置    →  多因子评分选股 或 指数择时+风控
```

### 调优方法

1. 默认参数跑5年完整回测
2. 观察收益来源是趋势还是震荡
3. 每次只改1个参数，对比变化
4. 找到局部最优后在 **样本外区间**（如最近1年）验证
